Tenants (Org containers)

A customer/org space in your portal (e.g., a USDA agency, state department, contractor org, university partner).

  • Each tenant has its own subdomain (e.g., myorg.agriportal.com).
  • Each tenant has its own branding (logo, colors, etc.).
  • Each tenant has its own admin users who can manage users and roles within that tenant.
  • Each tenant can have multiple users with different roles (e.g., admin, editor, viewer).
  • Teant owns survey projects, forms, datasets, users, and roles scoped to that org.
  • User = a person who signs in and belongs to one (or more) tenants.
